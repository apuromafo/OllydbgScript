这个脚本只能脱 exe, 只试过 Enigma v1.5x, v1.60, v1.61 加壳的程序. Enigma API 须自己搞因我懒得去看.

必须配合

1.Ollydbg 1.1
2.Odbgscript 1.64 或以上的版本
3.StrongOD 2.3
4.Import Reconstructor

                                    **使用前的修改**

把压缩包中的 Enigmapatch2.bin 和 Enigmavm.bin 放在一个你指定的地方, 然后用记事本搜寻  D:\  把他们改成你的路径 (共有 4 处须修改), 然后存档.


---------------------------------------------------------------------
Only support exe file which packed with Enigma v1.5x, v1.60, v1.61, you will need to fix Enigma API yourself (if it exist).

tools:

1.Ollydbg 1.1
2.Odbgscript 1.64 or above
3.StrongOD 2.3
4.Import Reconstructor



                            **Modification needed before usage**

Copy the Enigmapatch2.bin and Enigmavm.bin into a folder you want , then use text editor search for  D:\  , replace the path with where you put these two files (total 4 lines to be modified) .
